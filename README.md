# Detail Emotion 
> Emotion display percent > 60 แสดงอารมณ์ความรู้สึกเมือมีความรู้สึกนั้น มากกว่า60 %
- anger โกรธ 
- contempt กำลังดูถูกคน
- disgust รังเกียจ
- fear กลัว 
- happiness มีความสุข 
- neutral นิ่งเฉย 
- sadness โศกเศร้า 
- surprise เซอร์ไพร์
